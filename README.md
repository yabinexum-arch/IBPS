# IBPS - Sistema de Gestión Académica

Sistema de Gestión Académica para el Instituto Bíblico Patricio Symes. Esta es una **PWA (Progressive Web App)** diseñada para funcionar de manera offline y ser instalable en dispositivos móviles y de escritorio.

## Características
- Gestión de Semestres, Materias y Estudiantes.
- Registro de Parciales, Trabajos y Notas.
- Control de Asistencias y Disciplina.
- Generación de Historial Académico y Reportes Finales.
- Almacenamiento local mediante `localStorage`.
- Funcionalidad Offline mediante Service Workers.

## Instalación
1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en cualquier navegador moderno.
3. Para instalarlo como aplicación, haz clic en el icono de instalación en la barra de direcciones del navegador.
